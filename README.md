## Prerequisites
* Node.js
* Meteor
    - *Windows* -  Download and Install [Meteor](https://www.meteor.com/)
    - *Linux/Mac* - `curl https://install.meteor.com/ | sh`

## Execution
* Clone
    - git clone https://github.com/naver-d2-suji/suji-mr.git
    - cd suji-mr
    - git checkout feature/BoWoon
    - meteor
